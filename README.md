# House Price Prediction using Flask & Machine Learning

A web-based application that predicts house prices based on property details and city using a machine learning model.

---

## Tech Stack
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML, CSS, Bootstrap
- JavaScript

---

## Features
- User-friendly web interface
- Dynamic city selection based on state
- ML-powered price prediction
- RandomForest Regression model

---

## Machine Learning Model
- Algorithm: RandomForestRegressor
- Input Features:
  - Area (sqft)
  - Bedrooms
  - Bathrooms
  - City (encoded)
- Target: House Price
